## [0.1.0] - 2026-04-29

### Added

- Dark and light themes with independent temperature (`cool`, `neutral`, `warm`)
  and contrast (`soft`, `balanced`, `crisp`) axes
- Configurable cursor color, selection color, italic comments and italic
  keywords
- Runtime theme regeneration on settings change
- Web extension support
